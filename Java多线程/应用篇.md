## 应用篇

### 效率

- sleep效率降低cpu使用率

  在while(true)中，程序执行会使cpu使用率急剧升高，可以通过sleep方法降低cpu的使用率**踩油门踩刹车**

  ```java
  public static void main(String[] args) {
          while (true){
              try {
                  TimeUnit.MICROSECONDS.sleep(1);
              } catch (InterruptedException e) {
                  e.printStackTrace();
              }
          }
      }
  ```
